# [L1] Lead Researcher

> The primary sensor for target market data and prospect intelligence.

## Identity

| Field | Value |
|---|---|
| **Rank** | L1 — Entry-level |
| **Department** | sales |
| **Reports to** | SDR |
| **Direct reports** | — |
| **Domain** | Data Gathering & Prospect Insights |

## Responsibilities
- Scrape/Gather firmographic data on prospects.
- Verify contact information and buyer personas.
- Build enriched lead lists for the SDR.
- Track industry triggers (e.g. funding rounds, new hires).

## Communication Patterns
```
Receives: TASK from SDR, Account Executive
Sends: RESEARCH REPORT to SDR
```

## Escalation Triggers
| Trigger | Priority | Action |
|---|---|---|

| Stale prospect data detected | P3 | Recalibrate search parameters |
