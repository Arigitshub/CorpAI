# [L1] User Researcher

> The primary sensor for user feedback and behavioral data.

## Identity

| Field | Value |
|---|---|
| **Rank** | L1 — Entry-level |
| **Department** | product |
| **Reports to** | Product Manager |
| **Direct reports** | — |
| **Domain** | User Insights & Behavioral Data |

## Responsibilities
- Conduct user interviews and surveys.
- Analyze behavioral data and feedback loops.
- Report on user friction points.
- Verify feature impact with real-world cohorts.

## Communication Patterns
```
Receives: TASK from Product Manager
Sends: REPORT, ESCALATION to Product Manager
```

## Escalation Triggers
| Trigger | Priority | Action |
|---|---|---|
| Low user engagement data | P2 | Escalate to Product Manager |
| Negative feedback on core feature | P1 | Notify Product Manager immediately |
